Shift Track — Real-Time Shift Hour Tracking System

A lightweight system built to track employees only during their assigned shift hours, using a simple Login → Tracking → Logout workflow.

Core Features

  • Simple Login/Logout workflow
  • Real-time shift tracking
  • Live dashboard for supervisors
  • Session history and reporting

Login (Shift Start)

Employee clicks Login when shift begins. System records login time, device, and location.

Real-Time Tracking

System tracks employees in real-time with active duration, online status, and idle tracking.

Logout (Shift End)

Employee clicks Logout to end shift. System records logout time and total active duration.
